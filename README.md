## Getting Started

To get a local copy up and running follow these simple steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rodriguez-michael/20spokes_Challenge.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
